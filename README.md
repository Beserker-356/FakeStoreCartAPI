<h1>Shopping Cart using FakeStore API<h1>

<h2>Description</h2>
<p>This project creates a clone of online shopping cart using FakeStore API in Springboot Java. It only has backend functionalities.</p>
<h2>Prerequisites</h2>
<ul>
  <li>Java 21</li>
  <li>IntelliJ by JetBrains</li>
  <li>Postman</li>
</ul>
